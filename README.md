## Installation

Install Cypress via npm: [cypress](https://cypress.io/) to install de

```bash
npm install cypress --save-dev
```

via yarn:


```bash
yarn add cypress --dev
```

run:

test run in headless
```bash
npn run test
```

test with report junit
```bash
npm run test:report
```

delete report 
```bash
npm run delete:report
```
